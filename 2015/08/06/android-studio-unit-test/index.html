看到一篇博文，介绍了如何在Android Studio中进行单元测试，很实用，特此转载。

我也在Android开发者官网上查询了测试，但是基础的讲解都是使用的Eclipse ADT或者命令行界面，对于Android Studio如何搭建测试环境等等没有找到。

Source: 使用Android Studio进行单元测试 - 赵凯强 - 博客频道 - CSDN.NET
注意：下面是对转载文章的补充。

假如你的项目中有一个类，定义如下：

public class ClassA {
    public static void method() {
        //codes here
    }
}

如果想要为该类的method方法创建单元测试用例，那么可以创建类methodTest(即代替文章中提到的ExampleTest类),其定义如下：

public class methodTest extends InstrumentationTestCase {
    public void runMethod() throws Exception{
        //test codes here
    }
}

上述的抛出异常的语句必须有，否则测试用例运行会失败。
