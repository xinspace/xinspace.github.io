用u盘安装操作系统

发表于 2013-03-30 | 分类于 Others

自从用了linux之后，各种发行版出的时络绎不绝啊！所以，如果想要在物理机上安装个个版本尝尝鲜的话，还要刻N多的碟子（我大一的时候就刻了6张碟子，惨啊～），极为不方便。

比起刻碟子，用u盘安装操作系统方便，环保还节约（最起码不用买N张空碟子，而且刻碟子多了对本机的光驱也是有损害的）。鉴于这个特点，我决定放弃光碟，改用u盘来安装各种操作系统。

将操作系统（windows或linux发行版都可以）写入u盘最为启动设备的步骤如下：

1.下载linux live usb creator（简称lili），网址:http://www.linuxliveusb.com/

注意哈，这个lili现在只有windows版本的，也就是你必须再windows下才能使用这款软件，这款软件可以将操作系统写入到u盘，将u盘制作成启动设备。虽然lili的全称时linux live usb creator，但并不意味着只能写linux操作系统，windows操作系统也能识别，只不过linux操作系统（若含有live版）比windows多了live mode。

2.安装lili，出现如下图所示的界面（可以说lili做的很漂亮的说）：

可以看到安装步骤共又5步（在虚拟机中截图的，没截完全）。

3.将操作系统写入u盘：

第一步：选择要作为启动设备的u盘等移动设备（这个应该都会）

这个lili软件比较有意思，如果你每一步的选择都可行的话，它右边的红绿灯就会亮绿灯，如果不可行就亮红灯，如果有些小欠缺就会亮黄灯。

第二步：选择镜像文件

在你的电脑上选择你要写入的操作系统镜像文件（iso,img等文件，如win7.iso），一般选择第一个选项即ISO/IMG/ZIP这个选项，但前提是你电脑上已经存在一个操作系统的镜像文件；当然如果你正好有一张碟子，上面已经刻上了操作系统的话，就要选第二个CD图标；同时，你也可以指定一个网站，直接从网上下载镜像安装到u盘中（当然，这仅局限于你的网速很不错）。

不管你用上面三种的哪种方式选中了系统镜像文件，lili都会检查该镜像是否可用，稍微等待一下即可。

第三步：选择系统镜像的模式类型

如果你要安装的系统镜像带有live模式（如fedora18，ubuntu 12.10等linux发行版都带有live mode，即可以在不安装系统的情况下体验系统，若觉得不错就可以马上安装，若觉得不好就退出体验而不会真正安装），则这一步你可以选择往u盘里写入的模式类型为live mode还是install only。顾名思义，install only就是没有体验模式，只进行安装。一般我会第三步：选择系统镜像的模式类型选择安装live mode。

如果你要安装的系统镜像没有live模式，如windows7的镜像，那么就只有isntall only 模式了。

第四步：一般默认会选中第一个和第三个选项，建议保持默认。至于中间那个选项，我也建议你选中。

为什么呢，因为只有你的u盘等移动设备时FAT32的文件系统的时候才能用作启动设备，所以你在写入操作系统之前最好把该u盘格式化成FAT32格式的。

如果你的u盘有10G，但是一般用作启动设备顶多占用2G，剩下的8G又不能随便动（如果u盘已经做成了启动设备，你还往里放东西就可能破坏之前写入的操作系统和相关文件，所以一般u盘用作启动设备之后就不要再往里放东西了），那就白白浪费了吗？当然不是，你可以用分区工具将这个10G u盘分成两个分区，其中一个分区分2G左右空间，专门用这个分区来作为启动设备即可。

第五步：开始写入。

如果没有特殊需要，直接点击那个闪电按钮即可，如果有些小洁癖，可以点击options按钮来设置一些选项。设置完了，就点击闪电按钮，然后耐心等待就行了。

等写入完成后，你的u盘（或者说指定的分区）就已经成为了启动引导设备了，就可以用这个u盘来安装操作系统了。快去试试吧。
