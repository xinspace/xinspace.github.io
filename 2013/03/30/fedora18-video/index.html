fedora18自带的video视频播放器不能播放视频(rmvb和avi)和音频(mp3)

发表于 2013-03-30 | 分类于 Others

装了fedora18之后，就一直努力解决自带的video视频播放器不能播放视频音频。

用video打开视频之后，总会弹出提示框说需要xxxx插件，是否搜寻（search），然后点击search之后，又说找不到。我了个去，这不是要命嘛。

在我的不懈努力下，终于还是解决了这个问题啦～

步骤如下：

1.在命令行输入：su -c ‘yum localinstall –nogpgcheck http://mirrors.163.com/rpmfusion/free/fedora/rpmfusion-free-release-stable.noarch.rpm http://mirrors.163.com/rpmfusion/nonfree/fedora/rpmfusion-nonfree-release-stable.noarch.rpm‘

你可别老老实实的一个字母一个字母的照着敲这行命令哈，直接复制粘贴就ok啦，可能复制的话引号会变成中文的，可以自己改一下，不然会出错！

输入完成后，点击回车会让你输入密码，这个密码是root的密码，而不是自己的登陆密码哈！

之所以我们点击搜寻（search）按钮后说搜寻失败，是因为我们本地软件源所对应的服务器上没有相应的插件而已，我们只要加入一个含有这些插件的服务器（软件源）即可，所以，上面这条命令就是在我们本地加入163这个服务器（软件源），这个服务器上有我们需要的视频和音频解码插器

2.安装音频支持，在命令行输入：sudo yum install gstreamer-plugins-good gstreamer-plugins-bad gstreamer-plugins-ugly libtunepimp-extras-freeworld xine-lib-extras-freeworld

3。安装视频支持，在命令行输入：sudo yum install ffmpeg ffmpeg-libs gstreamer-ffmpeg libmatroska xvidcore

4.到此，我们的视频音频源的问题就解决了。当我们再用video打开视频后，再search时，能够找到相应的插件了，而且我们还能install（安装）它了，马上我们就能看视频了。但是，问题come again！（如果你此时没有出现任何问题，那你就别往下看了，直接看视频去吧～哈哈）

5.这时可能会弹出另一个提示框，说让我们提交这个错误报告，什么错误呢，一大串英文看的迷迷糊糊的，错误提示如下：

GPG key retrieval failed: [Errno 14] Could not open/read file:///etc/pki/rpm-gpg/RPM-GPG-KEY-rpmfusion-nonfree-fedora-18-i386，大概的意思是打不开这个KEY文件，这时，万能的百度…

6.百度一下之后，明白了。原来，每 个 fusion源发布的稳定 RPM 软件包都配有一个 GPG 签名。默认情况下，yum 和图形更新工具在安装软件的时候会验证这些签名并拒绝安装任何没有签名或者签名损坏的软件包。

如下所述能够让你更明白：您总是应该在安装软件包之前验证其签名。这些签名可确保您要安装的软件包出自 fusion仓库，且没有被提供该软件包的网页或者镜像更换（无意的或者恶意的）。

但是，我们安装fusion源的时候，默认是没有添加gpg密钥的，所以就会出现说不能打开KEY这种错误。

找到病根了，对症下药就ok啦，我们加入密钥就ok！

7.打开fusion源官网上的密钥页面：http://rpmfusion.org/keys

往下拉滚动条，找到【RPM Fusion nonfree for Fedora 18】字样（因为本人使用的是fedora18，所以是free for fedora18），如下图所示：

这就是害我们不能看视频的密钥，我们终于找到了。点击【Download】，然后跳入另一个页面，努力不懈的再点击【Download】，就会下载这个密钥文件啦，我下载的文件名为：RPM-GPG-KEY-rpmfusion-free-fedora-18

8.打开终端（快捷键ctrl+alt+t,如果不行的话看本博的设置打开终端快捷键博文），将刚刚下载的文件拖到终端里，就会出现该文件的路径，然后将光标调到该行的最前面，输入: sudo rpm –import 就可以了

最终命令行上的命令效果是：sudo rpm –import ‘/home/xin/Downloads/RPM-GPG-KEY-rpmfusion-free-fedora-18′其中文件名可能不一样，根据自己下载的文件的不同而不同，回车之后，我们的密钥就加入成功了。

9.现在用video打开视频，再search插件，再install，就不会出现任何问题啦～费尽千辛万苦终于能看视频和听音频啦～现在感觉还是linux mint用起来方便阿，装上系统就能看电影听歌的，不像fedora啊，整死人。不过fedora比较适合开发，而且我用linux mint用烦了。

不罗嗦了，大家赶紧去看视频吧！
